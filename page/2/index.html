比赛经验

发表于 2019-11-07 更新于 2019-11-09 分类于 Others

前言

最近想了想,之前考试总是有一些奇奇怪怪的问题,所以决定总结一下考试经验,以备不时之需.

NOIp/CSP级别的比赛

这类比赛一般都比较简单,也不会开$subtask$,数据较水.
比较好的做题策略:

  1. 看完所有题.
  2. 再看一遍所有题,确保没有看错题.
  3. 解决所有一眼可以秒掉的题目.
  4. 对拍
  5. 思考较难题
  6. 考试还剩1.5h,花0.5h写完难题的暴力.
  7. 解决难题直至下考前5min.
  8. 检查文件名与是否删除调试信息.

注意事项:

  1. 对于较难题可以数据分治.
  2. 一定要对拍!!!

省选/WC/NOI级别比赛

题目较难,数据不是很水.
做题策略:

  1. 花$30min$看完所有题目.
  2. 标出你第一眼能够打的分数(一定要合理)
  3. 打完所有题的暴力.
  4. 每道题目花10min再想想能否突破.
  5. 大致写完所有题目.
  6. 对于难题进一步思考.
  7. 只要能够写出暴力就一定要对拍,你要想想造数据写拍并不麻烦.
  8. 注意常数优化

注意事项:

  1. 珍惜时间,不要浪费.
  2. 对拍造的数据不能太随机,要仔细构造.
  3. 不要在简单题上丢分,一定要拿到自己打的所有分,谨防$FST$.

希望自己能够有更进一步的突破吧!!!

B

考虑令每一种情况为$A$必胜,$B$必胜和$C$必胜,如果我们知道$C$的必胜情况,那么就可以判断剩下的.
因为$C$在必败的情况下一定会选择去帮助$A$然后达成自己次解的情况.
所以可以枚举每一堆石头减去的量,然后这就是$C$帮助$A$的量,最后$check$即可.

C

不用思考发现这个操作可逆,考虑将两个状态转移到一个相同的然后判断.
我们不妨钦定这个相同的就是满足$1$个数最少的同时第一个$1$尽可能靠前.

仔细思考发现,几次操作可以看成将一个$1$向前移动$k$位.
因为让一段$0$消失一定不优,所以我们只需要考虑$1$向前移动然后删除一段1.
你删除一段1就相当于他们对$k$取模是连续的一段,这个时候可以用一个vector存下$1$跳跃然后直接模拟即可.

「JOI 2017 Final」焚风现象

简单题,考虑区间修改只会对$l$和$r$产生影响,直接树状数组暴力修改即可.

「JOI 2017 Final」准高速电车

每一次用堆维护能够为答案增加最多贡献的位置点然后不断向后移动即可.

「JOI 2017 Final」JOIOI 王国

考虑选的只有可能是左上角,右下角,右上角,左下角四个位置,直接做4次二分即可.

「JOI 2017 Final」足球

$\texttt{copy from M_sea}$
首先可以发现最优解大概是这样一个过程：

  • 控球的球员运球。
  • 控球的球员踢球。
  • 离球最近的球员移动，然后控球。

因此可以把每个位置 $(x,y)$ 拆成 $3$ 个点，分别代表控球、东西方向滚动、南北方向滚动。

连边的话控球到滚边权为 $B$，滚到控球边权为 $dis_{x,y}\times C$（$dis_{x,y}$ 表示离 $(x,y)$ 最近的球员到 $(x,y)$ 的距离），运球到相邻格子边权为 $C$，滚到相邻格子边权为 $A$。

然后直接跑最短路即可。
