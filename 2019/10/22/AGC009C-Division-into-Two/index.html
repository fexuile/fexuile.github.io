AGC009C Division into Two

发表于 2019-10-22 分类于 Solution

传送门

Atcoder

$\Theta(n^2)$

Solution

很显然有一个$n^2$的$dp$，即：
令$f_{i,j}$表示，当前枚举到了第$i$个数，$A$集合中最大的元素是$a_i$，$B$集合中最大的元素是$a_j$，$g_{i,j}$的定义类似。
那么很明显可以写出转移：
$$
f_{i,j} \Longrightarrow
\begin{cases}
f_{i+1,j}\ \ \ s_{i+1}-s_i \ge A \newline
g_{i+1,i}\ \ \ s_{i+1}-s_j \ge B
\end{cases}
$$

$$
g_{i,j} \Longrightarrow
\begin{cases}
g_{i+1,j}\ \ \ s_{i+1}-s_i \ge B \newline
f_{i+1,i}\ \ \ s_{i+1}-s_j \ge A
\end{cases}
$$

Code

$\Theta(nlogn)$

Solution

这个时候发现可以把这个主动转移变成被动转移，然后就可以滚掉第一维是吧。
接着我们发现这个维护其实就是一个树状数组的形式，然后直接用树状数组维护即可。
清空数组的方式大致就是，因为你清空的东西一定递增（下标），所以每一次把上一次的位置到现在这里清空即可。

Code

$\Theta(n)$

Solution

考虑现在我们钦定$A<B$，那么可以设$f_{i}$表示$A$集合中结尾的数是$i$，这个怎么转移？
$f_{i}=\sum_{(j+1\sim i-1)的数差\ge A,s_i-s_j \ge B}$
然后我们发现$j$的范围是一段区间，直接前缀和随便搞搞即可。
最后统计答案记得不能统计多了。
