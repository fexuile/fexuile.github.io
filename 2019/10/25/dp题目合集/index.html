dp题目合集

发表于 2019-10-25 分类于 Solution

csp2019dp复习.

UVA10559 Blocks

神仙题,可以很显然的看出是区间$dp$,但是二维的状态无法考虑消除中间一段的贡献再合并两边,所以多设一维状态表示与$r$颜色相同的后面有多少个消除了(在原序列可以不连续,但是最后消除中要连续),直接转移即可.

[NOI2009]二叉查找树

不难看出题目要你构造的就是一棵$Treap$,这同时也是一棵二叉查找树,所以中序遍历是固定的,所以我们可以在中序遍历上$dp$,每一次选取一个为根然后合并左右最后深度全部+1即可.

CF10D LCIS

奇怪的题目…序列$dp$,转移位置直接每一次记下即可.

CF55D Beautiful numbers

数位$dp$入门题,$dfs$即可,多记一位$0/1$表示有没有达到上限.

[HAOI2008]硬币购物

简单题,直接$dp$完全背包出来,最后容斥即可.

[HNOI2012]集合选数

套路题,可以构造一个向右$\times 2$,向下$\times 3$的矩阵,然后状压即可,复杂度是$log_3n$和$log_2n$选一个状压的复杂度.

Luogu3959 宝藏

状压$dp$,枚举起始点即可.(据说会有一些奇怪的小问题)

[HNOI2008]玩具装箱TOY

斜率优化模板题,后续会开一个斜率优化的总结.

[JSOI2011]柠檬

同上

Luogu1850 换教室

概率$dp$模板题,设$f_{i,j,0/1}$表示前$i$课,申请了$j$次,上一次申请成功与否,转移即可.

Luogu1941 飞扬的小鸟

看退役前的做题记录1.0.

[HNOI/AHOI2018]道路

$dp$就思维上是入门题,但是空间开不下,注意每一个点相关的点不多,直接存一个栈记录即可.

[APIO2014]连珠线

神仙题,很显然可以枚举根节点,这样子你的转移方式就是唯一的,然后设$f_{u,0/1}$表示$u$为根的子树,$u$作为/不作为中间节点的最大值.总复杂度$O(n^2)$的.
接着考虑每一次只有儿子之间的转移,可以换根$dp$,具体来说就是设$dp_{u,0/1,j}$表示$f_{u,0/1}$不考虑$j$这个儿子的答案,存下最大值和次大值转移即可.

CF1039D You Are Given a Tree

首先我们注意对于一个$mid$而言,我们可以求出它的答案,那么对于$mid$左边的有一段区间,右边的也有一段区间,按照整体二分的那套理论做就行了.
