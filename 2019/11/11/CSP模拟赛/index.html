A 签到题

很简单的签到题,考虑写成杨辉三角的形式,就不难发现这是$(a+b)^n[m]$的值了.

B 还是签到题

确实还是签到题/jk.
考虑将所有的$B$缩在一起,那么显然有我们每一次可以钦定一段不选,然后其他的带权是吧.
这个东西可以动态维护,这样子就做到了$O(n)$的了.
代码

C 全是签到题

不得不说还是我对于直径了解不够透彻.
考虑合并两个连通块,直径显然只有两种去向:

  • 不变
  • 变化
    考虑不变就是原本大的直径,变化就是$dis_{u}+1+dis_{x}$,这里用$dis_{i}$表示离$i$的最远点的距离.
    这样子就很容易想到说可以开个桶维护了.
    复杂度分析的话:
  1. 对于每一个连通块大小$siz \le \sqrt{n}$的显然正确.
  2. 对于$siz \ge \sqrt{n}$的考虑记忆化,这样子所有$siz \ge \sqrt{n}$的最多有$n$次询问,这样子复杂度就没有问题了.
